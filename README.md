# 重庆大学千里战队机械组网站模板

这是机械组展示页面的第一版多页面静态模板，当前使用灰色色块作为图片和图表占位。

## 本地预览

直接双击 `index.html` 即可在浏览器中查看主页，并通过顶部导航进入各个子页面。

如果后续改成多页面或需要本地服务器，再补充更完整的启动方式。

## 页面结构

- `index.html`：首页
- `about.html`：关于我们
- `projects.html`：研发项目
- `resources.html`：实验室资源
- `team.html`：团队成员
- `alumni.html`：校友去向
- `join.html`：加入我们
- `contact.html`：联系方式

## 后续替换内容

- 队徽或战队 Logo
- 机器人照片、加工照片、实验室照片
- 研发项目的真实年份、负责人、技术亮点和结果
- 培养计划图、设备资源清单、课程知识地图
- 成员信息和校友去向
- 招新渠道和联系方式
